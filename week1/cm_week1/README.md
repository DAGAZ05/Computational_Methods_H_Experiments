## 问题概述

### 问题1：级数求和计算
计算 $S_n=\sum_{j=2}^{n}\frac{1}{j^2-1}$，比较从小到大和从大到小两种求和顺序的数值精度。

**关键发现**：从大到小求和数值更稳定，但差异在单精度下不明显。

### 问题2：高斯消去法求解线性方程组
使用高斯消去法求解含小参数ε的线性方程组，分析ε→0时的数值行为。

**关键发现**：方程组条件数稳定在2.618，属于良态问题，高斯消去法稳定。

### 问题3：无穷级数计算e^x
使用泰勒级数计算指数函数，比较正负x值的计算稳定性。

**关键发现**：负x值直接计算不稳定，使用 $e^x = 1/e^{-x}$ 公式可解决。

### 问题4：定积分递推计算
计算定积分 $y_n=\int_{0}^{1}x^n e^{x-1}dx$，比较正向和反向递推的数值稳定性。

**关键发现**：正向递推不稳定（误差指数增长），反向递推稳定（误差缩小）。

## 数值计算主题
通过这些实验，探讨了以下数值计算主题：
1. 浮点数精度与舍入误差
2. 算法数值稳定性

4. 误差传播分析
5. 稳定算法设计
